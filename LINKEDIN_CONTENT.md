🏥 **ICD-10 Search Engine: Complete Medical Reference Platform**

Proud to present **ICD-10 Search Engine** — a comprehensive medical database platform combining drug reference, medical coding, and condition information in one powerful tool.

### Platform Overview:

🔍 **Three Core Modules:**

**1. Drug Lens** — Smart medication search engine
- 8,141+ medications with complete profiles
- Find exact alternatives (same drug, same pharmaceutical form)
- Access dosages, interactions, pregnancy categories, black box warnings
- Arabic & English support

**2. ICD-10 Code Search** — Medical coding made easy
- 39,128+ ICD-10 diagnostic codes
- Quick lookup by code or description
- Organized by medical categories
- Essential for healthcare billing & documentation

**3. Condition Browser** — Medical conditions database
- 808+ medical conditions with detailed information
- Related medications & treatment options
- ICD-10 code associations
- Evidence-based information

### Why It Matters:
Healthcare professionals spend hours searching across multiple sources. This platform consolidates everything — medications, diagnoses, and codes — in one intuitive interface. Faster decisions, better patient care.

### Key Features:
✅ Advanced search (Google Style, Trade Name, Scientific Name, ICD-10 codes)
✅ Smart alternatives matching with inline accordion display
✅ Complete drug information (interactions, dosages, warnings)
✅ Integrated medical coding reference
✅ Dark mode for extended use
✅ Mobile-responsive design
✅ Real-time search results
